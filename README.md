## ⚡ What It Does

NotebookLM reads **your** documents and generates content from them — no internet, no hallucinations. Upload PDFs, YouTube links, websites, or text, and let AI do the heavy lifting.

---

## 📂 What You Can Upload

`PDF` · `YouTube URLs` · `Websites` · `Images` · `Audio` · `MP4` · `Plain Text`

> Free: **50 sources** &nbsp;|&nbsp; Paid: **300 sources**

---

## 🎯 Key Features

| Feature | What it does |
|---|---|
| 🎙️ **Audio Overview** | AI podcast from your docs — join live as a 3rd speaker |
| 🎬 **Video Overview** | Auto-generated explainer video (2–10 min) with custom visual styles |
| 🗺️ **Mind Map** | Interactive visual tree — click any node for a deep dive |
| 📊 **Slide Deck** | 10-slide presentation, fully revisable |
| 🃏 **Flashcards & Quiz** | Study mode — flip cards, track scores, test yourself |
| 🖼️ **Infographic** | One-image visual summary of all your content |
| 🤖 **Shareable Chatbot** | Share your notebook as a chat-only link for your team |

---

## 🚀 Quick Start

```
1. Visit notebooklm.google.com
2. Sign in with Google → Create New Notebook
3. Upload your sources
4. Open Studio → pick a feature → Generate
```

---

## 🔌 Bonus: YouTube Chrome Extension

Import entire playlists or channels in one click.

```
Search → "YouTube to NotebookLM Extension" → Add to Chrome
→ Open any playlist → Click the NotebookLM button → Done ✓
```
